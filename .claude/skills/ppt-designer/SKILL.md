# ppt-designer — 비즈니스 PPT 디자인 스킬

## 트리거
- "PPT 만들어줘", "슬라이드 만들어줘", "프레젠테이션 만들어줘"
- "ppt-designer 써서", "/ppt-designer"

## 역할
정해진 디자인 시스템을 기반으로 모던·프로페셔널 비즈니스 PPT 슬라이드를
HTML 또는 python-pptx 코드로 생성한다.

---

## 디자인 시스템

### 톤 & 무드
```
모던 / 프로페셔널 / 미니멀
정보 전달이 빠른 / 깔끔한 비즈니스 리포트 느낌
```

### 컬러 팔레트

#### 배경
| 용도 | 색상 | HEX |
|------|------|-----|
| 슬라이드 배경 | 화이트 | `#FFFFFF` |
| 보조 배경 / 캔버스 | 라이트 그레이 | `#F0F0F0` ~ `#F5F5F5` |
| 구분선 / 표 라인 | 연회색 | `#D9D9D9` |

#### 텍스트
| 용도 | 색상 | HEX |
|------|------|-----|
| 헤드라인 (메인) | 거의 블랙 | `#222222` |
| 본문 | 다크 그레이 | `#555555` |
| 보조 / 캡션 | 미디엄 그레이 | `#888888` ~ `#999999` |

#### 강조
| 용도 | 색상 | HEX |
|------|------|-----|
| 브랜드 오렌지 | 비비드 오렌지 | `#E24D26` |
| 차콜 포인트 | 진회색 | `#3F3F3F` |

---

### UI 포인트 요소 (필수 적용)

#### 1. 섹션 타이틀 앞 "+" 마크
```
+ 섹션 제목
  ─────────
```
- 색상: `#E24D26` (브랜드 오렌지)
- 폰트 웨이트: Bold
- 크기: 헤드라인과 동일

#### 2. 하단 우측 오렌지 사각 배지
```
슬라이드 우하단 고정
┌──────┐
│  01  │  ← 페이지 번호 또는 라벨
└──────┘
```
- 배경색: `#E24D26`
- 텍스트: `#FFFFFF`
- 크기: 40×40px (정사각형)
- 위치: right: 40px, bottom: 30px

#### 3. 오렌지 리본/바 캡션 영역
```
████████████████████████████████
  사진 캡션 또는 강조 문구
████████████████████████████████
```
- 배경색: `#E24D26`
- 텍스트: `#FFFFFF`
- 높이: 44px
- 용도: 이미지 하단 캡션 / 인용구 강조

│   [보조 배경 블록 좌측]                 │

│     서브타이틀 / 날짜                   │
│                                        │
│                          ┌────┐        │
│                          │ 01 │        │
│                          └────┘        │
└────────────────────────────────────────┘
```

### Type B — 콘텐츠 슬라이드 (텍스트)

│  ─────────────────────────────────────│
│                                        │

│  • 본문 항목 2                         │
│  • 본문 항목 3                         │

│                          ┌────┐        │
│                          │ 02 │        │
│                          └────┘        │
└────────────────────────────────────────┘
```

### Type C — 2단 레이아웃
```

│  + 섹션 제목                           │
│  ─────────────────────────────────────│
│  ┌──────────────┐  ┌──────────────┐   │

│  │  (텍스트/   │  │  (이미지/    │   │

│                          ┌────┐        │
│                          │ 03 │        │
│                          └────┘        │

```

### Type D — 데이터/표 슬라이드
```
┌────────────────────────────────────────┐
│  + 섹션 제목                           │
│  ─────────────────────────────────────│
│  ┌──────┬──────┬──────┬──────┐        │
│  │ 헤더 │ 헤더 │ 헤더 │ 헤더 │        │
│  ├──────┼──────┼──────┼──────┤        │
│  │      │      │      │      │        │
│  │      │      │      │      │        │
│  └──────┴──────┴──────┴──────┘        │
│  * 보조 설명 (캡션 색상)               │
│                          ┌────┐        │
│                          │ 04 │        │
│                          └────┘        │
└────────────────────────────────────────┘
```

## 출력 형식

### HTML 슬라이드 (기본)
- 1080×608px (16:9) 또는 1280×720px
- 인라인 CSS 완전 적용
- 위 디자인 시스템 100% 반영

### python-pptx 코드 (요청 시)
```python
# 예시 구조
from pptx import Presentation
from pptx.util import Inches, Pt
from pptx.dml.color import RGBColor

ORANGE = RGBColor(0xE2, 0x4D, 0x26)
CHARCOAL = RGBColor(0x3F, 0x3F, 0x3F)
HEADLINE = RGBColor(0x22, 0x22, 0x22)
BODY = RGBColor(0x55, 0x55, 0x55)
CAPTION = RGBColor(0x88, 0x88, 0x88)
BG_GRAY = RGBColor(0xF0, 0xF0, 0xF0)
DIVIDER = RGBColor(0xD9, 0xD9, 0xD9)
```

## 슬라이드 생성 원칙

1. **1슬라이드 1메시지** — 한 장에 하나의 핵심만
2. **여백 충분히** — 패딩 최소 40px 이상
3. **오렌지는 포인트만** — 전체 면적의 10% 이하
4. **텍스트 3단계 위계** — 헤드라인 > 본문 > 캡션
5. **배지는 항상 우하단 고정** — 모든 슬라이드 공통
6. **구분선은 #D9D9D9** — 헤드라인 아래 항상 삽입
7. **"+" 마크는 #E24D26** — 섹션 타이틀 앞 필수
