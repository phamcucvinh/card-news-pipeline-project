# Platform-Specific Viral Copywriting Formulas

## YouTube

### 제목 공식
- [숫자] + [결과] + [시간] : "3분 만에 SNS 팔로워 1000명 늘리는 법"
- [금지/비밀] : "절대 알려주지 않는 유튜브 알고리즘의 비밀"
- [before vs after] : "구독자 0명 → 10만명: 내가 한 것들"

### 설명란 공식
첫 2줄이 핵심 (접힘 위):
```
[핵심 가치 한 줄] 🔥
[궁금증 유발 한 줄]

⏰ 목차:
00:00 인트로
...

📌 관련 링크:
```

## Instagram

### 캡션 공식
- 훅 (1-2줄): 스크롤 멈추게 하는 첫 문장
- 본문 (3-5줄): 가치 제공
- CTA: "저장해두세요 📌", "아는 친구 태그"
- 해시태그: 대형(1M+) 3개 + 중형(100K) 10개 + 소형(10K) 17개

### 한국어 훅 패턴
- "솔직히 말할게요..."
- "이걸 모르면 손해예요"
- "저만 알고 싶었는데..."
- "충격받은 사실:"

## Threads

### 스레드 구조 (7개)
1. 훅 + 주장 (가장 강한 클레임)
2. 문제 제시
3. 해결책 예고
4. 핵심 포인트 1
5. 핵심 포인트 2
6. 핵심 포인트 3
7. 요약 + CTA

### Threads 특성
- 대화체, 구어체 사용
- 이모지 1-2개/스레드
- 마지막에 질문으로 마무리

## KakaoTalk

### 오픈채팅 공유 공식
```
💡 [핵심 한 줄]

[2-3줄 본문]

👇 더 보기
[링크]
```

## LinkedIn

### 전문가 포스트 공식
- 첫 줄: 통계 or 경험담으로 시작
- 3-5개 bullet point로 가치 제공
- 개인 인사이트 1-2문장
- CTA: "팔로우", "댓글로 의견 공유"
- 해시태그: 3-5개 (업계 관련)

## Naver Blog

### SEO 최적화
- 제목: 메인 키워드 포함, 25자 이내
- H2 소제목: 2-3개, 키워드 포함
- 본문: 1500자 이상, 이미지 3장 이상
- 마무리: 관련 포스트 링크
- 태그: 20개 (정확한 검색어 위주)
