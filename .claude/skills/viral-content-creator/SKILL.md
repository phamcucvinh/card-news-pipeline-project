# Viral Content Creator

Transform any input content into platform-optimized viral posts across multiple channels simultaneously.

## Supported Platforms

| Platform | Format | Optimal Length |
|----------|--------|---------------|
| YouTube | 제목 + 설명 + 태그 | 제목 60자, 설명 200자 |
| Instagram | 캡션 + 해시태그 | 150자 + 30태그 |
| Threads | 연속 스레드 | 500자/스레드, 5-7개 |
| KakaoTalk | 오픈채팅 공유용 | 200자 이하 |
| LinkedIn | 전문가 포스트 | 700자 |
| Naver Blog | SEO 최적화 글 | 1000자 이상 |

## Workflow

### 1. 입력 분석
- 콘텐츠 유형 파악: 아이디어/기사/영상/연구/경험
- 핵심 메시지 1-3개 추출
- 타겟 독자 파악
- 감정적 훅 요소 식별

### 2. 바이럴 요소 적용
Apply these viral triggers to every piece of content:
- **호기심 갭**: "당신이 모르는 X의 비밀"
- **숫자/리스트**: "5가지", "3단계" 등 구체적 수치
- **긴급성**: 시의성 있는 트렌드 연결
- **사회적 증거**: 데이터, 사례, 성과 수치
- **감정 자극**: 놀라움, 공감, 영감

### 3. 플랫폼별 생성
Use references/platform-hooks.md for platform-specific copywriting patterns.

For each platform requested:
1. 제목/훅 라인 3개 작성 (A/B 테스트용)
2. 본문 작성 (플랫폼 최적 길이)
3. CTA(행동 유도) 포함
4. 해시태그/키워드 추가

### 4. 출력 형식
Output each platform in a clearly labeled section with copy-paste ready content:

```
## [플랫폼명]
**훅 옵션 A**: ...
**훅 옵션 B**: ...
**본문**: ...
**해시태그**: ...
```

## Quick Commands

- "이 내용 바이럴로 만들어줘" → 모든 플랫폼 동시 생성
- "유튜브용으로 바꿔줘" → YouTube 전용
- "인스타 캡션" → Instagram 최적화
- "스레드 써줘" → Threads 연속 포스트
