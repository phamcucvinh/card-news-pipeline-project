# poem-writer — 시 집필 스킬

## 역할
확정된 시상과 형식을 바탕으로 시 초고를 집필한다.

## 입력
- `output/concepts/poem-concept-[제목].md` (시상 문서)
- 형식 지정 (자유시 / 시조 / 소네트 등)
- `feedback.md`

## 출력
`output/drafts/poem-[제목]-[YYYYMMDD].md`

---

## 집필 구조

```markdown
# [제목]

[시 본문]

---

**형식**: [자유시 / 시조 / 소네트 등]
**행수**: [N]행
**핵심 이미지**: [이 시의 중심 이미지]
**작가 노트**: [집필 의도 1-2문장]
```

---

## 형식별 집필 지침

### 자유시
- 호흡 단위로 행을 나눈다
- 연(聯) 구분은 의미 전환 시
- 첫 행과 마지막 행에 가장 강한 이미지 배치

### 시조 (3장 구조)
```
초장: [3·4·3·4 또는 3·4·4·4 음절]
중장: [3·4·4·4 음절] — 시상 전개
종장: [3·5·4·3 음절] — 반전 또는 여운
```
- 종장 첫 음보는 반드시 3음절

### 소네트 (14행)
- 1-8행: 문제 제기 / 상황
- 9-12행: 전환 또는 반론
- 13-14행: 결론 또는 반전

### 산문시
- 단락 1-3개
- 문장 끝에 운율보다 이미지 밀도를 높인다
- 마지막 문장이 시의 핵심

---

## 이미지 생성 원칙

| 감정 | 직접 표현 (금지) | 이미지 표현 (권장) |
|------|--------------|----------------|
| 그리움 | "그립다" | 다 식은 국, 열지 않은 편지, 빈 신발 |
| 슬픔 | "슬프다" | 비 맞은 종이, 꺼진 불, 돌아선 어깨 |
| 기쁨 | "기쁘다" | 갑자기 가벼워진 발걸음, 열린 창 |
| 외로움 | "외롭다" | 혼자 켜진 전등, 2인분 그릇 |

## 초고 작성 태도
- 완벽한 시는 퇴고에서 나온다 — 지금은 이미지를 쏟아낸다
- 행이 너무 길면 나누고, 너무 짧으면 합친다
- `[TODO: 이 행 재작성]` 표시 후 넘어가도 된다
