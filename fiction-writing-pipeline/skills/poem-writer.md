# poem-writer — 시 집필 스킬

## 역할
확정된 시상과 형식을 바탕으로 시 초고를 집필한다.

## 입력
- `output/concepts/poem-concept-[제목].md` (시상 문서)
- 형식 지정 (자유시 / 시조 / 소네트 등)
- `feedback.md`

## 출력
`output/drafts/poem-[제목]-[YYYYMMDD].md`

---

## 집필 구조

```markdown
# [제목]

[시 본문]

---

**형식**: [자유시 / 시조 / 소네트 등]
**행수**: [N]행
**핵심 이미지**: [이 시의 중심 이미지]
**작가 노트**: [집필 의도 1-2문장]
```

---

## 형식별 집필 지침

### 자유시
- 호흡 단위로 행을 나눈다
- 연(聯) 구분은 의미 전환 시
- 첫 행과 마지막 행에 가장 강한 이미지 배치

### 시조 (3장 구조)
```
초장: [3·4·3·4 또는 3·4·4·4 음절]
중장: [3·4·4·4 음절] — 시상 전개
종장: [3·5·4·3 음절] — 반전 또는 여운
```
- 종장 첫 음보는 반드시 3음절

### 소네트 (14행)
- 1-8행: 문제 제기 / 상황
- 9-12행: 전환 또는 반론
- 13-14행: 결론 또는 반전

### 산문시
- 단락 1-3개
- 문장 끝에 운율보다 이미지 밀도를 높인다
- 마지막 문장이 시의 핵심

## 이미지 생성 원칙

| 감정 | 직접 표현 (금지) | 이미지 표현 (권장) |
|------|--------------|----------------|
| 그리움 | "그립다" | 다 식은 국, 열지 않은 편지, 빈 신발 |
| 슬픔 | "슬프다" | 비 맞은 종이, 꺼진 불, 돌아선 어깨 |
| 기쁨 | "기쁘다" | 갑자기 가벼워진 발걸음, 열린 창 |
| 외로움 | "외롭다" | 혼자 켜진 전등, 2인분 그릇 |

## 초고 작성 태도
- 완벽한 시는 퇴고에서 나온다 — 지금은 이미지를 쏟아낸다
- 행이 너무 길면 나누고, 너무 짧으면 합친다
- `[TODO: 이 행 재작성]` 표시 후 넘어가도 된다

## 커스텀 스타일 레퍼런스 — *Remember* 스타일 (2026.02.26 등록)

> 이 스타일은 실제 작성된 시 *Remember*의 분석 결과를 바탕으로 커스텀 등록된 참조 규칙이다.
> 요청 시 `--style remember` 플래그로 적용한다.

### 구조 패턴
```
[1연] — 수용 / 이별 / 놓아줌       (2–3행)
[2연] — 자기 성장 / 역설적 전환    (3–4행)
[3연] — 집착과 무상 / 이미지 절정  (2–3행)
[결론] — 중도(中道) 선언            (1–2행)
총 목표: 여백 포함 400–450자 내외
```

### 어조 규칙
- **1인칭 고백이 아닌 보편 선언** — "나는 슬프다" ❌ → "슬픔은 계절이다" ✅
- 조용하되 흔들리지 않는 목소리 — 설교가 아닌 확언
- 독자가 '자신의 이야기'로 읽을 수 있는 열린 문장

### 이미지 규칙
| 주제 | 직접 표현 (금지) | Remember 스타일 표현 |
|------|--------------|-------------------|
| 이별 | "그가 떠났다" | "Everything has its season" |
| 집착 | "놓지 못한다" | "The more you cling… the further it drifts" |
| 성장 | "힘들었지만 성장했다" | "the ground you grew from" |
| 위로 | "괜찮아질 것이다" | "do not blame yourself" |

### 문체 규칙
- **단언형(declarative)** 문장 위주 — 의문문 최소화
- **enjambment 의도적 사용** — 끊김 자체가 의미가 됨
- **구체 → 추상** 흐름: 사람/계절 같은 구체 이미지 → 행복/슬픔 같은 보편 진리
- **미니멀리즘** — 형용사는 꼭 필요할 때만, 한 행 = 하나의 완결된 생각

### 철학적 베이스
- 무상(無常) · 비집착 — 불교적 수용
- 통제 가능한 것에 집중 — 스토아 철학
- 역경은 성장의 토양 — 성장 철학
- 결론은 이항 대립 구조로 마감: "no eternal happiness, and no eternal sorrow"

### 퇴고 체크리스트 (Remember 스타일)
- [ ] 마지막 행이 이미지 또는 선언으로 끝나는가?
- [ ] 같은 단어가 두 번 이상 반복되지 않는가?
- [ ] 모든 형용사가 필수인가? (없애도 의미가 살면 제거)
- [ ] 소리 내어 읽었을 때 막히는 행이 없는가?
- [ ] 여백 포함 글자수가 400–500자 범위인가?
