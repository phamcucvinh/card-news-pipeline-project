# character-editor — 캐릭터 편집 스킬 (2차 편집)

## 역할
구조 편집이 완료된 원고에서 인물의 일관성, 동기, 관계 역학을 점검하고 수정한다.

## 입력
- 구조 편집 완료된 초고 (전체)
- `output/characters/characters-[제목].md`
- `output/drafts/structural-notes-[제목].md` (1차 편집 메모)
- `feedback.md`

## 출력
- `output/drafts/character-notes-[제목]-[YYYYMMDD].md` (편집 메모)
- 수정된 원고 (각 챕터 파일 업데이트)

---

## 캐릭터 편집 체크리스트

### 주인공
- [ ] 각 챕터에서 주인공의 동기가 일관되는가?
- [ ] Want와 Need의 긴장감이 유지되는가?
- [ ] 변화 궤적이 점진적이고 설득력 있는가?
- [ ] 캐릭터 설정(characters.md)과 충돌하는 행동이 없는가?

### 조력자/적대자
- [ ] 적대자의 동기가 이해 가능한가?
- [ ] 조력자가 단순 도구로 전락하지 않는가?
- [ ] 각 인물의 말투 특성이 유지되는가?

### 관계 역학
- [ ] 인물 관계의 변화가 자연스럽게 쌓이는가?
- [ ] 갈작스러운 관계 변화 없음 (감정 전환 근거 있음)?

### 대화 일관성
전체 대화를 각 인물별로 추출해서:
- [ ] 인물별 말투 고유성이 유지되는가?
- [ ] 같은 인물이 상황에 따라 너무 다르게 말하지 않는가?

---

## 편집 메모 형식

```markdown
# 캐릭터 편집 메모: [제목]
날짜: YYYY-MM-DD

## 주요 수정 사항

### [인물명]
- 챕터 [N]: [문제] → [수정 방향]

### 대화 수정
- 챕터 [N], 씬 [M]: "[기존 대사]" → [수정 이유]

## 다음 단계 (문장 편집) 주의사항
- [캐릭터 편집 후 문장 편집자가 알아야 할 사항]
```
