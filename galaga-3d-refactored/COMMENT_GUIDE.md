# 코드 주석 가이드

## 완료된 파일
✅ config.js - 모든 게임 설정값과 상수
✅ SceneManager.js - Three.js 씬 관리

## 주석 추가 예정 파일
- Player.js - 플레이어 클래스
- Enemy.js - 적 클래스  
- EnemyManager.js - 적 관리자
- BulletManager.js - 총알 관리자
- CollisionManager.js - 충돌 감지
- UIManager.js - UI 관리
- Game.js - 메인 게임 로직

각 파일에는 다음이 포함됩니다:
1. 파일 목적 설명 (헤더 주석)
2. 클래스 설명
3. 생성자 설명
4. 메서드별 상세 설명
5. 매개변수와 반환값 설명
6. 주요 로직 단계별 주석
