## 스토리텔링 글쓰기 콘텐츠 구성

### 1. 스토리텔링 기초
- [ ] 스토리텔링이란 무엇인가
- [ ] 왜 스토리는 사람을 움직이는가 (뇌과학적 근거)
- [ ] 좋은 이야기의 3요소 (캐릭터, 갈등, 해결)
- [ ] 사실(Fact)과 이야기(Story)의 차이

### 2. 이야기 구조
- [ ] 3막 구조 (발단 → 전개 → 결말)
- [ ] 영웅의 여정 (Hero's Journey) 12단계
- [ ] 피라미드 구조 (결론 먼저 말하기)
- [ ] IN MEDIAS RES (중간부터 시작하기) 기법

### 3. 캐릭터 만들기
- [ ] 매력적인 주인공 설정법
- [ ] 독자가 공감하는 캐릭터 만들기
- [ ] 결함 있는 캐릭터의 힘
- [ ] 실제 인물(고객, 나 자신)을 캐릭터로 활용하기

### 4. 감정을 자극하는 글쓰기
- [ ] 감정 곡선 설계법
- [ ] 긴장감과 기대감 만드는 법
- [ ] 독자가 몰입하게 만드는 디테일 묘사
- [ ] 공감 언어 사용법

### 5. 브랜드 스토리텔링
- [ ] 창업 스토리 / 브랜드 탄생 이야기 작성법
- [ ] 고객 성공 사례(케이스스터디) 스토리 작성법
- [ ] 브랜드 미션 / 비전을 이야기로 전달하는 법
- [ ] "Before & After" 스토리 구조 활용법

### 6. 콘텐츠별 스토리텔링 적용
- [ ] 블로그 / 아티클에 스토리 입히기
- [ ] SNS 포스팅에 스토리텔링 활용법
- [ ] 영상 스크립트 스토리 구성
- [ ] 프레젠테이션 / 피치덱 스토리 구조
- [ ] 광고 카피에 스토리 녹이기

### 7. 논픽션 & 자기계발 스토리
- [ ] 개인 경험을 콘텐츠로 만드는 법
- [ ] 실패 스토리를 자산으로 바꾸는 법
