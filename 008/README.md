## 보고서 글쓰기 콘텐츠 구성

### 1. 보고서 글쓰기 기초
- [ ] 보고서란 무엇인가 (목적과 역할)
- [ ] 좋은 보고서의 조건 (정확성, 간결성, 논리성)
- [ ] 보고서와 기획서의 차이
- [ ] 독자(상사/의사결정자) 중심으로 쓰는 법

### 2. 보고서 구조와 목차

- [ ] 피라미드 구조 (MECE 원칙 적용)
- [ ] 보고서 목차 설계법
- [ ] 요약문(Executive Summary) 작성법

### 3. 보고서 유형별 작성법

- [ ] 결과 보고서 작성법
- [ ] 현황 분석 보고서 작성법
- [ ] 제안 / 건의 보고서 작성법

- [ ] 사고 / 장애 보고서 작성법
