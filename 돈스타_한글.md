## 게임 개요

Don't Starve는 신비로운 야생에 갇힌 과학자 윌슨(Wilson)의 역할을 맡는 생존 게임입니다. 목표는 간단하지만 도전적입니다: **최대한 오래 생존하기**.

**핵심 원칙**: "승리"는 없습니다 - 오직 생존만이 있을 뿐입니다. 죽음은 영구적이며(로그라이크), 죽으면 처음부터 다시 시작해야 합니다.

## 핵심 게임 메커니즘

### 세 가지 핵심 스탯

**1. 체력(Health)** ❤️
- 받을 수 있는 피해량을 나타냄
- 0에 도달 = 죽음 또는 부활(부활 아이템이 있는 경우)
- 회복 방법: 음식, 치료 아이템, 수면

**2. 배고픔(Hunger)** 🍖
- 시간이 지나면서 지속적으로 감소
- 배고픔이 비면 = 지속적인 체력 손실
- 회복 방법: 모든 음식 아이템

**3. 정신력(Sanity)** 🧠
- 감소 원인: 어둠, 날고기 섭취, 몬스터와의 전투
- 낮은 정신력 효과: 화면 왜곡, 적대적인 그림자 생물 출현
- 회복 방법: 익힌 음식, 의복 아이템(실크햇), 꽃 따기

### 낮/밤 사이클
- **낮**: 탐험과 자원 수집에 안전
- **황혼**: 어둠 전의 짧은 기간
- **밤**: 극도로 위험 - 찰리(보이지 않는 몬스터)가 어둠 속에서 공격
- 밤에는 항상 광원 확보 필요(모닥불, 횃불 등)

## 1일차: 첫 단계

### 즉각적인 우선순위

**1. 기본 자원 수집** (첫 30분의 낮 시간)
```
✓ 부싯돌(Flint) - 도구용, 땅에서 주움
✓ 나뭇가지(Twigs) - 묘목에서, 12개 이상 필요
✓ 자른 풀(Cut Grass) - 풀 덤불에서, 12개 이상 필요
✓ 통나무(Logs) - 도끼로 나무 베기, 10개 이상 필요
✓ 음식(베리, 당근) - 최대 5개
```

**2. 필수 도구 제작**
- **도끼(Axe)**: 부싯돌 1개 + 나뭇가지 1개 (나무를 베기 위해)
- **곡괭이(Pickaxe)**: 부싯돌 2개 + 나뭇가지 2개 (바위 채굴용)

**3. 밤 준비**
- **모닥불(Campfire)**: 통나무 2개 + 풀 3개 (어두워지기 전에 건설!)
- **횃불(Torches)**: 풀 2개 + 나뭇가지 1개씩 (밤 탐험용)

**4. 음식 조리**
- 음식 아이템을 불 근처에 놓아서 조리
- 익힌 음식 = 더 나은 배고픔 회복 + 더 긴 유통기한

### 1일차 금지사항
❌ 거미, 벌, 톨버드 공격하지 말 것
❌ 늪 생물군계에 들어가지 말 것
❌ 아직 영구 건축물 짓지 말 것 (이동할 예정)
❌ 음식을 너무 많이 모으지 말 것 (상함)
❌ **굶지 마세요!**

## 2-7일차: 초반 게임

### 주요 목표

**2-3일차: 탐험과 정찰**
- 최적의 기지 위치를 찾기 위한 탐험
- 자원과 생물군계 지도 작성
- 도로 찾기 (중요 지역 연결)
- **아직 정착하지 말 것** - 계속 이동

**4-5일차: 기지 선택**

다음 근처 위치 선택:
- ✅ 도로 교차로 (쉬운 이동)
- ✅ 토끼 구멍 2-3개 (안정적인 식량원)
- ✅ 연못 (개구리, 낚시)
- ✅ 돼지 마을 또는 버팔로 무리 (1화면 거리)
- ✅ 풀, 묘목, 베리 덤불

다음 근처 정착 피하기:
- ❌ 바다코끼리 캠프
- ❌ 박쥐가 있는 싱크홀
- ❌ 거미 둥지 (너무 가까움)

**6-7일차: 기지 설립**
```
우선순위 제작:
1. 과학 기계(Science Machine): 통나무 4개 + 바위 4개 + 금 덩어리 1개
2. 배낭(Backpack): 풀 4개 + 나뭇가지 4개 - 인벤토리 공간 증가
3. 통나무 갑옷(Log Suit): 통나무 8개 + 밧줄 2개 - 기본 방어구
4. 창(Spear): 나뭇가지 2개 + 밧줄 1개 + 부싯돌 1개 - 무기
```

### 자원 관리 팁
- 2-3일치 음식 비축
- 고기는 먹기 전에 모두 조리 (정신력 손실 방지)
- 초반 게임에서 금 낭비하지 말 것 (과학 기계용으로 저축)

## 8-20일차: 중반 게임

### 인프라 개발

**1. 과학 기계 잠금 해제**
```
필수 제작물:
- 배낭 (인벤토리 확장)
- 삽 (식물을 파서 이동)
- 망치 (구조물 해체, 돼지 가죽 획득)
- 화덕(Fire Pit) (영구 광원)
- 크록팟(Crock Pot) (고급 요리)
```

**2. 농장 설치**
- 삽으로 풀 덤불 파기
- 기지로 이동
- 거름으로 비료 주기 (돼지/버팔로에서)
- 묘목과 베리 덤불도 동일하게

**3. 식량 생산**
- 토끼 구멍 근처에 토끼 덫 6-10개 설치
- 고기 보존용 건조대 건설
- 씨앗으로 농장 만들기 (채집보다 비효율적)

**4. 광원**
- **랜턴(Lantern)**: 나뭇가지 2개 + 밧줄 2개 + 전구 2개 - 휴대용 조명
- **채광모(Mining Hat)**: 금 1개 + 밀짚모자 1개 - 양손 자유로운 조명

### 탐험 목표
- 터치스톤 찾기 (부활 지점)
- 웜홀 위치 파악 (빠른 이동)
- 모든 생물군계 지도 작성
- 계절 보스 위치 확인

## 21일차 이후: 겨울 준비

### 겨울 생존 필수품

**온도 관리**
```
겨울 장비:
- 보온석(Thermal Stone): 바위 10개 + 곡괭이 1개 + 부싯돌 3개
  • 열을 흡수하고 천천히 방출
  • 탐험 전에 불 근처에서 가열

- 겨울 모자(Winter Hat): 버팔로 털 4개
  • +120 단열
  • 열 손실 감소

- 푹신한 조끼(Puffy Vest): 버팔로 털 8개 + 버팔로 뿔 2개
  • +240 단열
  • 최고의 겨울 의류
```

**식량 비축**
- 겨울 전에 40개 이상의 음식 아이템 수집
- 크록팟 레시피에 집중 (음식 가치 배가)
- 고기 건조 (신선한 것보다 오래 지속)

**연료 비축**
- 통나무 60개 이상 비축
- 횃불용 풀과 나뭇가지 보관
- 연료용 버팔로 털 고려

### 겨울 사냥: 맥터스크(MacTusk)
- 겨울 동안 바다코끼리 캠프에 출현
- 귀중한 전리품 드롭:
  - 탐오샨터(Tam o' Shanter) (최고의 겨울 모자)
  - 지팡이(Walking Cane) (25% 속도 증가)
  - 블로우 다트(Blow Darts) (원거리 무기)

## 전투 가이드

### B.A.R. 방법

**카이팅 전략** (때리고 도망)
```
1. BAIT(유인): 적이 먼저 공격하게 함
2. AVOID(회피): 적의 공격을 피함
3. RETALIATE(반격): 쿨다운 동안 2-3회 타격
4. RETREAT(후퇴): 뒤로 물러나서 반복
```

### 적 난이도별 전투

**쉬움** (초보자 친화적)
- 거미(Spiders) (1회 타격 패턴)
- 개구리(Frogs) (약하고 느림)
- 고블러(Gobblers) (도망감)

**중간**
- 하운드(Hounds) (무리 공격)
- 촉수(Tentacles) (높은 피해, 고정)
- 트리가드(Treeguards) (탱크 적)

**어려움** (숙련될 때까지 피하기)
- 톱니바퀴 적들(Clockwork enemies) (기사, 주교, 룩)
- 거인들(Giants) (계절 보스)
- 그림자 생물(Shadow creatures) (낮은 정신력)

### 전투 장비
```
초보자용:
- 통나무 갑옷(Log Suit): 통나무 8개 + 밧줄 2개 - 80% 피해 감소
- 창(Spear): 나뭇가지 2개 + 밧줄 1개 + 부싯돌 1개 - 34 피해

고급:
- 풋볼 헬멧(Football Helmet): 돼지 가죽 1개 + 밧줄 1개 - 80% 감소
- 햄 배트(Ham Bat): 고기 6개 + 돼지 가죽 2개 + 밧줄 2개 - 59.5 피해
- 다크 소드(Dark Sword): 악몽 연료 5개 + 살아있는 통나무 1개 - 68 피해
```

## 음식과 요리

### 크록팟 기초

**요리의 이점**
- 음식 가치 배가 (1+1+1+1 = 날것으로 먹는 것보다 좋음)
- 부정적 효과 제거 (날고기 정신력 페널티)
- 음식이 오래 지속 (6-20일 vs 2-4일)

### 필수 레시피

**1. 미트볼(Meatballs)** (⭐ 초반 게임 최고)
- 레시피: 고기 1개 (모든 크기) + 필러 3개 (얼음, 베리, 버섯)
- 결과: 배고픔 +62.5, 체력 +5, 정신력 +5
- 지속: 10일

**2. 미티 스튜(Meaty Stew)**
- 레시피: 고기 3개 + 필러 1개
- 결과: 배고픔 +150, 체력 +12, 정신력 +5
- 지속: 10일

**3. 피에로기(Pierogi)** (⭐ 최고의 치유)
- 레시피: 고기 1개 + 달걀 1개 + 채소 1개 + 필러 1개
- 결과: 배고픔 +37.5, 체력 +40, 정신력 +5
- 지속: 20일

**4. 트레일 믹스(Trail Mix)** (채식 옵션)
- 레시피: 구운 자작나무 열매 1개 + 베리 1개 + 과일 1개 + 필러 1개
- 결과: 배고픔 +12.5, 체력 +30, 정신력 +5
- 지속: 15일

### 음식 보존 방법

**건조대(Drying Rack)**
- 고기를 육포로 변환
- 육포는 20일 지속 (익힌 고기 6일 대비)
- 정신력 회복 (+10-15)

**아이스박스(Ice Box)**
- 음식 부패를 크게 늦춤
- 필요: 금 2개 + 톱니바퀴 1개 + 판자 1개
- 효율성을 위해 크록팟 근처에 배치

## 계절별 생존

### 봄 (약 21-35일차)
**도전과제**: 비, 습기, 번개
- **우비**: 우산, 레인코트
- **피뢰침**: 기지를 화재로부터 보호
- **습기 효과**: 정신력 손실, 동결, 미끄러운 아이템

### 여름 (약 36-50일차)
**도전과제**: 과열, 산불, 거인
- **냉각 아이템**: 얼음 큐브, 보온석(냉각됨), 꽃무늬 셔츠
- **화재 예방**: 플링고매틱(자동 소화)
- **얼음 공급원**: 아이스박스, 겨울 준비 중 얼음 낚시

### 가을 (약 51-70일차)
**도전과제**: 겨울 준비
- **수확 시즌**: 최대 자원 수집
- **비축 구축**: 음식, 연료, 재료
- **쉬운 계절**: 기본 시작 계절과 유사

### 겨울 (약 71-90일차 이상)
**도전과제**: 동결, 식량 부족
- **따뜻하게 유지**: 보온석, 겨울 의류
- **식량원**: 코알라판트 사냥, 얼음 낚시
- **거인**: 디어클롭스(겨울 30.8일 이후 공격)

## 고급 팁

### 캐릭터 선택
**초보자 친화적:**
- 윌슨(Wilson) (기본, 단열용 수염 자람)
- 윌로우(Willow) (화재 피해 면역, 불에서 정신력 회복)
- WX-78 (톱니바퀴로 스탯 증가)

**숙련될 때까지 피하기:**
- 울프강(Wolfgang) (강력할 때 음식 페널티)
- 위그프리드(Wigfrid) (육식만 가능, 공격적 플레이스타일)
- 웨스(Wes) (가장 어려운 캐릭터 - 모든 스탯 페널티)

### 부활 옵션
**고기 조각상(Meat Effigy)**
- 비용: 돼지 가죽 4개 + 버팔로 털 4개 + 수염 털 4개
- 조각상 위치에서 부활
- 영구 최대 체력 페널티 (-30)

**생명을 주는 부적(Life-Giving Amulet)**
- 비용: 금 3개 + 붉은 보석 1개 + 악몽 연료 2개
- 착용 가능, 죽음 시 부활
- 영구 페널티 없음

**터치스톤(Touch Stone)**
- 세계에 자연 스폰
- 일회용 부활 지점
- 죽기 전에 찾아서 활성화

### 효율성 팁
1. **항상 음식 조리** - 더 나은 가치, 더 오래 지속
2. **모든 것을 카이팅** - 피해 입지 않기
3. **자원 근처에 기지** - 이동 시간 절약
4. **크록팟 레시피 사용** - 음식 효율성 배가
5. **계절 준비** - 마지막 순간까지 기다리지 말 것
6. **스폰 주기 학습** - 하운드 3-13일마다, 거인 계절별

### 흔한 실수
❌ 너무 일찍 건설 (잘못된 위치)
❌ 겨울 준비 안 함
❌ 방어구 없이 전투
❌ 날음식 섭취 (정신력 손실)
❌ 밤에 탐험 (횃불 연료 낭비)
❌ 초반 게임에서 한 곳에 너무 오래 머무름

## 빠른 참조 카드

### 일별 체크리스트
```
1일차: 자원 수집, 도구 제작, 모닥불 만들기
2-3일차: 탐험, 기지 위치 정찰
4-5일차: 기지 선택, 건축 재료 수집
6-7일차: 과학 기계 건설, 배낭/방어구 제작
8-14일차: 농장 설립, 덫 설치, 음식 수집
15-20일차: 겨울 장비 준비, 자원 비축
21일차 이후: 겨울 생존, 맥터스크 사냥
```

### 생존 우선순위
1. **음식** - 항상 2일 이상의 비축분
2. **빛** - 연료 없이 어둠에 갇히지 않기
3. **온기** - 온도 관리
4. **안전** - 방어구와 무기 준비
5. **정신력** - 30 이상 유지 (그림자 생물 회피)
