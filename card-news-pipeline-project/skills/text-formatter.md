# text-formatter — 텍스트 포맷팅 스킬

## 역할
원문을 슬라이드용으로 줄바꿈·강조 처리한다.
**원문 보존이 핵심** — 단어/내용 수정 절대 금지.

## 입력
- 원문 텍스트 (column.md 또는 JJ 직접 입력)
- 시리즈 유형 (A 또는 C)

## 출력
`output/columns/formatted-column-[주제]-[YYYYMMDD].md`

---

## 포맷팅 규칙

### 기본 규칙
- 슬라이드당 최대 **50자** (한글 기준)
- 강조어: `**볼드**` 처리
- 줄바꿈: 의미 단위로 분리 (문장 중간 자르기 금지)
- 슬라이드 구분: `---` 구분선 사용

### 강조 판단 기준
볼드 처리할 것:
- 핵심 키워드 (주제어)
- 숫자/데이터
- 액션 동사 (해야 한다, 활용하라 등)
- 대비 표현 (하지만, 반면에 등)

### C시리즈 특별 규칙
- ⚠️ **원문 단어 수정 절대 금지**
- 줄바꿈과 강조만 추가
- 원문의 문장 순서 변경 금지
- 오타도 그대로 보존 (JJ에게 확인 후 수정)

---

## 출력 형식

```markdown
# 포맷팅 완료: [주제/제목]
원문 글자수: XXX자 | 슬라이드 예상: XX장

---

[슬라이드 1 - 표지]
**[제목]**
[부제 또는 첫 문장]

---

[슬라이드 2]
[본문 내용]
**[강조 키워드]** [이어지는 내용]

---

[슬라이드 3]
...
```

---

## 슬라이드 분량 가이드

| 원문 길이 | 예상 슬라이드 수 |
|-----------|-----------------|
| ~500자 | 4-5장 |
| 500-1000자 | 6-8장 |
| 1000-1500자 | 8-10장 |
| 1500자~ | 10-12장 (긴 경우 요약 권장) |
