# hashtag-generator — 해시태그 생성 스킬

## 역할
칼럼/카드뉴스 주제와 채널에 맞는 최적 해시태그 세트를 생성한다.
노출 극대화를 위한 대·중·소 해시태그 믹스 전략 적용.

## 입력
- 칼럼 또는 카드뉴스 주제
- 채널 (인스타그램 / 링크드인 / 스레드)
- 시리즈 유형 (A/B/C/D)
- `feedback.md` (기존 해시태그 취향 참조)

## 출력
채팅으로 직접 제시 (저장 불필요)

---

## 채널별 해시태그 전략

### 인스타그램
- **총 개수**: 20-30개
- **구성**: 대형(100만+) 5개 + 중형(1-10만) 10개 + 소형(1만 이하) 10개
- **언어**: 한국어 + 영어 믹스
- **위치**: 캡션 하단 또는 첫 댓글

```
#마케팅 #디지털마케팅 #마케터 (대형)
#코딩하는마케터 #마케팅자동화 (중형)
#[시리즈명] #[주제키워드] (소형/브랜드)
```

### 링크드인
- **총 개수**: 5-10개
- **구성**: 업계 키워드 중심, 전문성 강조
- **언어**: 한국어 위주, 글로벌 키워드 2-3개

```
#마케팅전략 #AI마케팅 #디지털트랜스포메이션
#Marketing #GrowthHacking
```

### 스레드
- **총 개수**: 3-5개
- **구성**: 간결하고 트렌디한 키워드
- **언어**: 자연스러운 단어

## 시리즈별 고정 해시태그

| 시리즈 | 고정 해시태그 |
|--------|-------------|
| A (코딩하는 마케터) | #코딩하는마케터 #마케터코딩 #마케팅자동화 |
| B (알쓸IT잡) | #알쓸IT잡 #IT용어 #디지털리터러시 |
| C (커리어 레터) | #커리어 #직장인 #커리어성장 |
| D (뉴스레터) | #뉴스레터 #마케팅인사이트 #위클리 |

---

## 해시태그 작성 원칙
1. 브랜드 해시태그는 항상 포함 (시리즈 고정)
2. 클릭베이트성 태그 금지 (#선팔맞팔 등)
3. 주제와 무관한 인기 태그 금지
4. 채널별 알고리즘에 맞는 개수 준수
