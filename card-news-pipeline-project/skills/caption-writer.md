# caption-writer — SNS 캡션 작성 스킬

## 역할
카드뉴스 업로드 시 사용할 인스타그램/스레드 캡션을 작성한다.

## 입력
- 원고 파일 (column.md 또는 슬라이드 내용)
- 게시 플랫폼 (인스타그램 / 스레드 / 둘 다)
- `feedback.md`

## 출력
`output/captions/caption-[주제]-[YYYYMMDD].txt`

---

## 인스타그램 캡션 구조

```
[후킹 첫 줄 — 3초 안에 멈추게 하는 문장]

[공백 한 줄]

[본문 3-5줄]
핵심 내용 요약, 슬라이드 보고 싶게 만들기

[공백 한 줄]

[더보기 클릭 유도]
↓ 슬라이드에서 자세히 확인하세요

[공백 한 줄]

[해시태그 10-15개]
#마케터 #카드뉴스 #마케팅 #[주제태그] ...

[공백 한 줄]

💬 [참여 유도 질문 또는 CTA]
```

---

## 스레드 캡션 구조

```
[후킹 첫 줄 — 스레드는 더 짧고 직접적으로]

[핵심 2-3줄]

[해시태그 3-5개만]
```

---

## 해시태그 기본 세트

### 항상 포함
```
#마케터 #마케팅 #코딩하는마케터 #임지은
```

### A시리즈 추가
```
#칼럼 #마케팅칼럼 #AI마케팅 #디지털마케팅
```

### B시리즈 추가
```
#IT용어 #알쓸IT잡 #마케터공부 #IT공부
```

### C시리즈 추가
```
#커리어 #커리어레터 #직장인 #성장
```

---

## 후킹 첫 줄 패턴

- "마케터인데 이것도 모르면..." → 공감/긴장
- "3년 차에 깨달은 [주제]" → 경험/신뢰
- "[숫자]개만 알면 충분합니다" → 단순화/실용
- "솔직히 말하면, [반전]" → 궁금증
- "저도 처음엔 몰랐어요" → 공감/친근함
