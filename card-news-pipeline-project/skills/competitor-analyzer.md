# competitor-analyzer — 경쟁사 콘텐츠 분석 스킬

## 역할
유사 계정·경쟁사의 콘텐츠 전략을 분석하여 JJ 채널 전략에 활용할 인사이트를 도출한다.
벤치마킹을 통해 차별화 전략을 찾는 것이 목표.

## 입력
- 분석 대상 계정명 (인스타그램 / 링크드인 / 스레드)
- 분석 기간 (최근 N개월)
- 분석 목적 (콘텐츠 주제 / 포맷 / 해시태그 / 발행 전략)
- `feedback.md`

## 출력
`output/research/competitor-[계정명]-[YYYYMMDD].md`

---

## 분석 프레임워크

### 1. 콘텐츠 전략 분석
```markdown
## 계정명: [@계정]
분석 기간: [기간] | 총 게시물: [N]개

### 콘텐츠 믹스
| 형식 | 비율 | 평균 반응 |
|------|------|---------|
| 카드뉴스 | N% | 좋아요 N개 |
| 단일 이미지 | N% | |
| 영상/릴스 | N% | |
| 텍스트 | N% | |

### 주요 주제 TOP 5
1. [주제] — [빈도] — [평균 반응]
2. ...

### 발행 패턴
- 빈도: 주 N회
- 요일: [요일]
- 시간: [시간대]
```

### 2. 최고 성과 콘텐츠 분석
```markdown
### 반응 TOP 3
1. [제목/주제] — 좋아요 N | 댓글 N | 저장 N
   - 성공 요인: [분석]
2. ...
```

### 3. 차별화 기회 분석
```markdown
### 경쟁사가 다루지 않는 주제
- [주제 1]: [왜 기회인가]
- [주제 2]:

### JJ만의 차별화 포인트
- [차별화 요소 1]
- [차별화 요소 2]
```

---

## 분석 윤리 원칙
1. **공개 정보만 분석**: 공개된 게시물·반응만 수집
2. **벤치마킹 목적**: 모방이 아닌 차별화 전략 수립
3. **정기 업데이트**: 월 1회 분석으로 트렌드 파악
4. **주관적 평가 지양**: 수치 기반 객관적 분석

---

## 분석 대상 추천 카테고리
- 마케팅 교육 계정
- 코딩 입문 계정
- 직장인 커리어 계정
- 국내 뉴스레터 계정
- 해외 마케팅 인플루언서 (벤치마킹)
