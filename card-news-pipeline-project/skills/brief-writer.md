# brief-writer — 방향성 + 제목 작성 스킬

## 역할
리서치 결과를 바탕으로 콘텐츠 방향성과 후킹 제목 3안을 제시한다.
JJ가 쉽게 선택하고 수정할 수 있도록 근거와 함께 제안.

## 입력
- `output/research/research-[주제].md`
- `feedback.md` (JJ 선호 제목 스타일 참조)

## 출력 형식 (채팅으로 JJ에게 직접 제시)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 콘텐츠 방향성 + 제목 3안
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 콘텐츠 방향성
[2-3줄. 이 글이 독자에게 어떤 가치를 줄 것인지]

## 핵심 메시지 (한 줄)
[이 카드뉴스의 핵심을 한 문장으로]

---

## 제목 후보 3안

**1안** — [제목]
→ [왜 이 제목인지 1줄 설명]
→ 타겟 감성: [궁금증 자극 / 공감 / 실용성 / 충격]

**2안** — [제목]
→ [설명]
→ 타겟 감성: [...]

**3안** — [제목]
→ [설명]
→ 타겟 감성: [...]

---

## 🎯 추천 제목: [n]안
이유: [한줄 추천 이유]
```

---

## 제목 작성 원칙

### 구조적 원칙
- **20자 이내** 권장 (인스타그램 표지 가독성)
- 숫자 포함 시 클릭률 ↑ (예: "3가지", "5분 만에")
- 의문형 또는 단정형 중 하나 선택

### JJ 스타일 (feedback.md 기반 업데이트)
- 직설적이고 실용적인 표현
- 마케터 독자 타겟 의식
- 과장/클릭베이트 지양
- 1인칭보다는 독자 중심 ("당신의 ~", "마케터라면 ~")

### 피해야 할 제목 패턴
- ❌ "~하는 법 A to Z" (진부)
- ❌ "충격! ~" (클릭베이트)
- ❌ 30자 이상 (표지에서 잘림)

### 좋은 제목 예시
- ✅ "GPT로 칼럼 쓰는 마케터의 루틴"
- ✅ "API가 뭔지 몰라도 이건 알아야 해"
- ✅ "이직 전 반드시 확인해야 할 3가지"
