# validate-slide — 슬라이드 QA 검증 스킬

## 역할
완성된 HTML 슬라이드를 자동/수동으로 검증한다.
고/중 결함 발견 시 수정 후 재검증. 저 결함만 남으면 통과.

## 실행 방법
```bash
python scripts/validate_slide.py output/slides/
```

---

## 검증 항목

### 🔴 고(High) 결함 — 즉시 수정 필수

| 항목 | 검사 방법 | 기준 |
|------|-----------|------|
| 슬라이드 크기 | CSS width/height 파싱 | 정확히 1080x1080px |
| 텍스트 오버플로우 | overflow:hidden 확인 | 없어야 함 |
| 외부 리소스 | src/href 파싱 | 외부 URL 없어야 함 |
| HTML 유효성 | BeautifulSoup 파싱 | 파싱 오류 없어야 함 |

### 🟡 중(Medium) 결함 — 수정 권장

| 항목 | 기준 |
|------|------|
| 폰트 로드 | Pretendard 포함 여부 |
| 색상 대비 | 4.5:1 이상 (추정) |
| 빈 슬라이드 | 텍스트 없는 슬라이드 |
| 슬라이드 수 | 시리즈별 기준 (A:7-10, B:5-7, C:4-8) |

### 🟢 저(Low) 결함 — 참고용

| 항목 | 기준 |
|------|------|
| 줄당 글자수 | 50자 권장 초과 여부 |
| 이모지 사용 | 과다 사용 경고 (3개/슬라이드 초과) |
| 슬라이드 제목 | 파일명-내용 일치 여부 |
