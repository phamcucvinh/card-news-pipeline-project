# quote-extractor — 핵심 인용구 추출 스킬

## 역할
칼럼 또는 리서치 문서에서 카드뉴스 슬라이드·SNS 공유용 핵심 문장을 추출한다.
한 눈에 기억되고 공유하고 싶어지는 문장을 골라내는 것이 목표.

## 입력
- `output/columns/column-[주제].md`
- 용도 (카드뉴스 슬라이드 / 인스타 인용 / 링크드인 / 뉴스레터)
- `feedback.md`

## 출력
채팅으로 직접 제시

---

## 인용구 추출 기준

### 좋은 인용구의 조건
- [ ] 30자 이내 — 한 눈에 읽힌다
- [ ] 단독으로 의미가 완결된다 — 앞뒤 문맥 없이도 이해 가능
- [ ] 공감 또는 인사이트가 담겨 있다
- [ ] 행동을 유도하거나 질문을 던진다
- [ ] 숫자나 대비 구조가 있으면 더 좋다

### 인용구 유형
| 유형 | 예시 |
|------|------|
| **인사이트형** | "마케터의 반복 업무 62%는 자동화 가능하다" |
| **반전형** | "열심히 일하는 것보다 덜 일하는 설계가 낫다" |
| **행동 유도형** | "지금 당장 엑셀 매크로 하나만 배워보세요" |
| **공감형** | "저도 처음엔 코드 한 줄이 무서웠습니다" |

---

## 출력 형식

```
## 핵심 인용구 — [주제]

### 🥇 추천 1순위
> "[인용구]"
용도: [카드뉴스 표지 / 인스타 공유 / 뉴스레터 하이라이트]

### 🥈 2순위
> "[인용구]"

### 🥉 3순위
> "[인용구]"

### 추가 후보 (5개)
- "[인용구 1]"
- "[인용구 2]"
...
```

---

## 카드뉴스 슬라이드 특화 원칙
- 표지 슬라이드: 가장 강렬한 한 줄
- 중간 슬라이드: 소제목 대신 인용구 활용
- 마지막 슬라이드: 행동 유도형 인용구
